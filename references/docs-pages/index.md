# Dotty Foundation Docs Pages

Snapshot date: 2026-05-26

This folder mirrors the final Figma foundation documentation pages as curated Markdown for engineers, designers, and AI agents.

The `Foundations/` package is the current source-of-truth contract. Figma is the temporary visual and variable reference while the foundation is still being moved toward online documentation:
https://www.figma.com/design/WSSIyMUfJz9BsiVx5ocIL9/Dotty-Foundations

## How To Use

- Read `../Dotty Foundations Guide.md` first for the complete foundation contract.
- Use these files when you need page-level documentation without opening every Figma board.
- Use Figma links inside each file when visual layout, examples, swatches, diagrams, or exact board anatomy matters.
- Treat these files as part of the current foundation contract; use Figma only when visual inspection or variable inspection is needed.
